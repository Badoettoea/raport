# Raport Web App

Aplikasi web raport menggunakan Vue.js dan database Google Sheets.

## Fitur
- Login menggunakan PIN (dari Google Sheets)
- Dashboard User:
  - Lihat Profil
  - Upload Foto
  - Lihat Nilai Raport
- Dashboard Admin:
  - Upload Nilai
  - Download Leger CSV

## Teknologi
- Vue 3
- Vite
- Google Sheets API
- SheetDB (alternatif koneksi)
